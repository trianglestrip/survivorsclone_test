# 暖雪风格升级演示指南

## 如何测试改进

### 1. 运行自动化测试

```bash
godot --headless --script tests/test_stage1_warmsnow_upgrade.gd
```

或使用完整路径：

```bash
& "F:\project\godot\Godot_v4.6.1-stable_win64.exe" --headless --script tests/test_stage1_warmsnow_upgrade.gd
```

### 2. 在编辑器中测试

1. 打开 Godot 编辑器
2. 运行场景：`tests/test_stage1_operations.tscn` 或 `World/world.tscn`
3. 测试以下功能：

#### 近战攻击测试
- 按 **空格** 或 **鼠标左键** 进行攻击
- 观察：
  - ✓ 斩击特效是否流畅（8帧动画）
  - ✓ 是否有轻微屏幕震动
  - ✓ 攻击间隔是否约 0.3 秒
  - ✓ 命中敌人时是否有明显震动和停顿

#### 冲刺闪避测试
- 按 **Shift** 进行冲刺
- 观察：
  - ✓ 冲刺是否快速（0.12秒）
  - ✓ 是否有蓝色残影效果
  - ✓ 是否有轻微屏幕震动
  - ✓ 冷却时间是否约 0.8 秒
  - ✓ 技能栏Shift图标是否显示冷却数字

#### 受击反馈测试
- 让敌人攻击玩家
- 观察：
  - ✓ 是否有强烈屏幕震动
  - ✓ 角色是否短暂变红
  - ✓ 受击特效是否明显（红色爆发）
  - ✓ 血条是否平滑下降
  - ✓ 血条颜色是否根据血量变化（绿→黄→红）

#### 技能栏UI测试
- 观察屏幕左下角技能栏
- 检查：
  - ✓ 4个技能槽是否有发光边框
  - ✓ 就绪技能是否有脉冲效果
  - ✓ 冷却时是否显示倒计时数字
  - ✓ 冷却进度是否从上到下填充

#### 血条UI测试
- 观察屏幕顶部血条
- 检查：
  - ✓ 血条是否有边框和发光效果
  - ✓ 受击时是否闪白
  - ✓ 血量低于30%时是否红色脉冲
  - ✓ 数字显示是否清晰

### 3. 资源热重载测试

1. 在游戏运行时，修改任意UI或特效文件
2. 观察控制台输出：
   ```
   [ResourceHotReload] 检测到资源变化，重新加载:
     - res://Textures/UI/skill_slot_q.png
     ✓ 纹理已重载: skill_slot_q.png
   ```
3. 资源应自动刷新，无需重启

## 避免缓存问题的最佳实践

### 开发时
1. 确保 `ResourceHotReload` 已启用（已在 project.godot 中自动加载）
2. 修改资源后等待 1 秒，系统会自动检测
3. 如果未刷新，在控制台执行：
   ```gdscript
   ResourceHotReload.force_reload_all()
   ```

### 发布时
1. 在 `resource_hot_reload.gd` 中设置：
   ```gdscript
   @export var enable_hot_reload: bool = false
   ```
2. 或在 project.godot 中移除 ResourceHotReload 自动加载

### 手动清除缓存
如果遇到顽固的缓存问题：
1. 关闭 Godot 编辑器
2. 删除 `.godot/` 文件夹
3. 重新打开项目

## 性能基准

### 攻击系统
- 攻击冷却：0.3s（每秒最多 3.3 次攻击）
- 特效帧数：8帧 × 0.04s = 0.32s 总时长
- 屏幕震动：8次 × 0.02s = 0.16s 总时长

### 冲刺系统
- 冲刺持续：0.12s
- 冲刺冷却：0.8s
- 无敌时间：0.3s
- 残影生成：每 0.02s 一个（约 6 个残影）
- 残影淡出：0.3s

### UI系统
- 技能栏更新：每帧
- 血条平滑速度：5.0 lerp
- 发光脉冲频率：2.0 Hz

## 常见问题

### Q: 屏幕震动太强/太弱？
A: 修改配置文件中的 `screen_shake_intensity` 参数，或直接在代码中调整 `_shake_camera()` 的 `intensity` 参数。

### Q: 残影效果不显示？
A: 检查 `config/stage1_controls.json` 中 `trail_effect` 是否为 `true`。

### Q: UI没有刷新？
A: 执行 `ResourceHotReload.clear_godot_cache()` 或删除 `.godot/` 文件夹重启。

### Q: 冷却数字不显示？
A: 检查技能栏UI是否正确调用 `update_skill_cooldown()` 方法。
