3. 按F5或点击"运行场景"


## 🧪 测试流程建议

### 快速验证流程（5分钟）
1. **启动场景** → 观察初始状态
2. **按1** → 切换到冰心宗
3. **按Q/E/R** → 测试冰心宗3个技能
4. **按F** → 生成敌人测试伤害
5. **按2/3/4** → 切换其他宗派，重复步骤3-4
6. **按5-9** → 切换不同武器，测试攻击模式

### 完整测试流程（15分钟）
1. **宗派系统测试**（每个宗派）：
   - 切换宗派
   - 测试Q技能（观察弹幕/攻击）
   - 测试E技能（观察领域/范围效果）
   - 测试R技能（观察终极技能）
   - 验证技能特效是否显示
   - 验证技能伤害是否生效

2. **武器系统测试**（每种武器）：
   - 切换武器
   - 测试普通攻击（空格/左键）
   - 观察攻击速度差异
   - 观察攻击范围差异
   - 验证远程武器的弹道

3. **操作系统测试**：
   - 测试移动流畅度（WASD）
   - 测试冲刺手感（Shift）
   - 测试攻击响应（空格）
   - 测试受击反馈

## 🔍 重点观察项

### 视觉效果
- [ ] 技能特效是否显示？
- [ ] 特效颜色是否正确？（冰=蓝，雷=紫，火=橙，毒=绿）
- [ ] 特效是否在持续时间后消失？
- [ ] 击中特效是否触发？

### 功能性
- [ ] 技能是否造成伤害？
- [ ] 冷却时间是否正常？
- [ ] 武器切换是否生效？
- [ ] 宗派切换后技能是否改变？

### 手感
- [ ] 操作是否流畅？
- [ ] 是否有明显卡顿？
- [ ] 屏幕震动是否合适？
- [ ] 打击感是否足够？

## 🐛 常见问题

**Q：技能特效看不到？**
A：检查技能是否在冷却中，或者尝试切换宗派重新测试。

**Q：敌人不受伤？**
A：确保技能范围覆盖到敌人，可以按F生成更多敌人。

**Q：切换宗派/武器没反应？**
A：按键需要短按，不要长按。每次切换会在控制台打印确认信息。

**Q：场景崩溃？**
A：检查控制台错误信息，可能是某个系统初始化失败。
