# 暖雪风格技能系统规划

## 问题分析

### 当前技能 vs 暖雪风格

**当前技能（Survivors 风格）：**
- ❌ 冰刺（Ice Spear）- 远程魔法攻击
- ❌ 龙卷风（Tornado）- AOE 范围魔法
- ❌ 标枪（Javelin）- 远程投掷

**暖雪核心玩法：**
- ✅ 近战武器为主（剑、刀、枪等）
- ✅ 宗派系统（不同战斗风格）
- ✅ 主动技能 + 圣物加成
- ✅ 武器进化系统
- ✅ 闪避/冲刺作为核心操作

---

## 暖雪技能系统设计

### 1. 宗派系统（Sects）

#### 玄武宗（防御反击）
- **特点**：高防御，反击伤害
- **武器**：重剑、盾牌
- **技能**：
  - 铁壁：增加防御
  - 反击：受到攻击时反弹伤害
  - 震地：范围震击

#### 朱雀宗（火焰攻击）
- **特点**：高伤害，火焰特效
- **武器**：太刀、双刀
- **技能**：
  - 烈焰斩：火焰附魔攻击
  - 凤凰冲刺：燃烧路径
  - 爆发：范围火焰爆炸

#### 白虎宗（快速连击）
- **特点**：高攻速，连击加成
- **武器**：短剑、拳套
- **技能**：
  - 疾风步：增加移速攻速
  - 连斩：多段连击
  - 影遁：短暂隐身

#### 青龙宗（雷电控制）
- **特点**：控制效果，范围伤害
- **武器**：长枪、法杖
- **技能**：
  - 雷电附魔：攻击带电
  - 连锁闪电：多目标弹射
  - 雷暴：大范围雷击

### 2. 武器系统

#### 基础武器类型
1. **剑** - 平衡型，攻守兼备
2. **刀** - 高伤害，攻速稍慢
3. **枪** - 长距离，攻击范围大
4. **双刀** - 高攻速，连击加成
5. **拳套** - 极快攻速，短距离

#### 武器进化
- 普通 → 精良 → 史诗 → 传说
- 每次进化改变外观和属性
- 传说武器有特殊特效

### 3. 圣物系统（Relics）

#### 功能型圣物
- 增加攻击力
- 增加防御力
- 增加移动速度
- 增加攻击速度
- 增加暴击率

#### 特效型圣物
- 攻击吸血
- 受击回血
- 攻击时概率额外伤害
- 闪避时触发特效
- 击杀时恢复生命

### 4. 技能操作

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| 鼠标左键 / 空格 | 普通攻击 |
| Shift | 闪避/冲刺 |
| Q / E | 主动技能1 / 2 |
| R | 终极技能 |
