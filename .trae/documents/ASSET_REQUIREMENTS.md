# 暖雪风格资源需求文档

## 📋 资源清单总览

本文档详细描述所有需要绘制的技能特效和UI资源，包括尺寸、颜色、动画要求和文件命名规范。

**总计需求**：
- 技能特效：4组（Q/E/R技能 + 3种武器攻击）
- UI元素：15个（血条、状态栏、宗派界面等）
- 图标：20个（技能图标、武器图标、圣物图标）

## ⚔️ 第一部分：技能特效

### 1.1 Q技能特效 - 冰霜冲击（宗派技能1）

**描述**：向前方发射一道冰霜冲击波，冻结路径上的敌人

**动画帧数**：8帧  
**尺寸**：128x128 像素/帧  
**格式**：PNG，透明背景  
**颜色主题**：冰蓝色系

**分帧描述**：
- **帧0**：冰霜能量凝聚（小光点，20%大小）
- **帧1**：能量扩散（40%大小，半透明）
- **帧2**：冲击波形成（60%大小，开始有冰晶）
- **帧3**：冲击波爆发（80%大小，冰晶飞溅）
- **帧4**：冲击波全开（100%大小，冰晶环绕）
- **帧5**：冲击波扩散（110%大小，冰晶散开）
- **帧6**：冲击波减弱（90%大小，冰晶消散）
- **帧7**：冲击波消失（60%大小，淡出）

**视觉元素**：
- 中心：明亮的冰蓝色光球
- 周围：旋转的冰晶碎片（6-8个）
- 边缘：淡蓝色光晕（发光效果）
- 轨迹：冰霜粒子尾迹

**文件命名**：
```
Textures/Skills/Effects/FrostBlast/frost_blast_0.png
Textures/Skills/Effects/FrostBlast/frost_blast_1.png
...
Textures/Skills/Effects/FrostBlast/frost_blast_7.png
```

## 📐 通用设计规范

### 尺寸标准
- **小图标**：32x32 或 48x48（状态、圣物）
- **中图标**：64x64（技能、武器）
- **大图标**：96x96（宗派）
- **UI槽位**：80x80（技能槽）
- **面板**：320x240 或更大

### 透明度规范
- **实体元素**：100%不透明
- **发光效果**：30-60%透明度
- **背景**：85-95%透明度
- **遮罩**：70-80%透明度

### 边框规范
- **普通边框**：2px
- **强调边框**：3-4px
- **装饰边框**：6px
- **发光边框**：外扩2-4px

### 发光效果
- **内发光**：向内2px羽化
- **外发光**：向外2-4px羽化
- **强发光**：向外6-8px羽化

### 文字规范
- **小文字**：12-14号字体
- **中文字**：16-18号字体
- **大文字**：24-32号字体
- **描边**：2px黑色描边（所有文字）

## 🎨 绘制技巧建议

### 特效动画
1. **起始帧**：小而集中，能量凝聚
2. **中间帧**：逐渐扩大，效果全开
3. **结束帧**：淡出消散，粒子飞散
4. **运动模糊**：快速移动的帧添加轻微模糊
5. **发光层**：使用叠加模式增强发光感

### UI元素
1. **边框**：使用多层边框增强立体感
2. **发光**：使用外发光+内发光增强质感
3. **纹理**：添加细微纹理避免单调
4. **装饰**：四角或边缘添加小装饰
5. **渐变**：使用渐变而非纯色

### 图标设计
1. **主图案**：占据60-70%空间，清晰可辨
2. **背景**：圆形或方形，深色
3. **边框**：根据品质使用不同颜色
4. **发光**：主图案周围有发光效果
5. **细节**：添加小装饰增强精致感

## 🔧 技术规格

### 文件格式
- **格式**：PNG（推荐）或 WebP
- **色彩**：RGBA（必须支持透明度）
- **位深度**：32位（8位/通道）
- **压缩**：PNG-8 或 PNG-24

### 分辨率
- **像素密度**：72 DPI
- **缩放**：按实际像素尺寸绘制，不要过大
- **抗锯齿**：适度抗锯齿，保持像素风格

### 透明度
- **背景**：完全透明（Alpha = 0）
- **主体**：完全不透明（Alpha = 255）
- **发光**：半透明（Alpha = 100-200）
- **边缘**：渐变透明（羽化效果）

### 命名规范
- **小写字母**：全部使用小写
- **下划线分隔**：使用下划线而非空格或连字符
- **序号格式**：使用单个数字（0-9）或两位数字（00-99）
- **扩展名**：.png

**正确示例**：
- `frost_blast_0.png` ✅
- `thunder_strike_05.png` ✅
- `sect_icon_tianji.png` ✅

**错误示例**：
- `FrostBlast_0.png` ❌（大写）
- `thunder-strike-5.png` ❌（连字符）
- `sect icon tianji.png` ❌（空格）
