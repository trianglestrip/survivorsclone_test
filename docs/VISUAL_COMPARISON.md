## 🔧 技术实现细节

### 纹理加载双路径

```gdscript
// 路径1: 资源系统（生产环境）
if ResourceLoader.exists(path):
    texture = load(path)  // 使用.import优化

// 路径2: 直接加载（开发环境）
else:
    image = Image.new()
    image.load(abs_path)  // 不需要.import
    texture = ImageTexture.create_from_image(image)
```

**优势**:
- ✅ 开发时无需等待导入
- ✅ 支持热重载
- ✅ 生产环境自动优化
- ✅ 向后兼容

### 范围显示公式

```gdscript
// 通用公式
final_radius = base_radius * scale_factor * initial_scale * (growth_rate ^ frames)

// 雷霆一击示例
初始: 100 * 0.5 * 0.8 * 1.0 = 40px
最终: 100 * 0.5 * 0.8 * 1.05^5 = 51px

// 设计原则
scale_factor: 0.5-0.7  (显示为实际的50-70%)
initial_scale: 0.8-1.0 (初始稍小)
growth_rate: 1.05-1.1  (缓慢扩散)
frames: 5-8            (动画时长)
```

### 自动清理机制

```gdscript
// auto_cleanup_node.gd
extends Node

var cleanup_delay: float = 0.0
var fade_duration: float = 0.5

func _ready():
    if cleanup_delay > 0:
        await get_tree().create_timer(cleanup_delay).timeout
    
    # 淡出动画
    var parent = get_parent()
    if parent is CanvasItem:
        var tween = create_tween()
        tween.tween_property(parent, "modulate:a", 0.0, fade_duration)
        await tween.finished
    
    # 清理节点
    if is_instance_valid(parent):
        parent.queue_free()
```
